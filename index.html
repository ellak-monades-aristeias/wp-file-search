wp-file-search

WP File Search is a powerful extension (plugin) for Wordpress content management system which enables searching on the content of the pdf, docx and odt files that are uploaded to the Wordpress media library.

Requirements

It is required that you install the Wordpress platform, before installing the wp file search extension. The development of the plugin has been done with Apache (2.2), PHP (>=4) and MySQL (>=5.5). The plugin will probably be functional also with earlier versions.

Branches

There are two active branches from where you can download the code of the extension:

  • master branch, which contained a stable, tested plugin version. The content of the master branch is available in the wordpress plugin directory.
  • development branch, where the current version of the plugin is included with the latest bugs corrected and new features added. The development branch is periodically merge with the master. Any other branch is experimental and its use is discouraged.

Wiki

The wiki includes detailed installation and usage instructions. Additional information is provided for those who wish to expand the application features.

Issues

You can upload new issues with the appropriate label:

  • Use the label enhancement to mention a new feature that you would like to be added.
  • Use the label bug to report a functional problem. The label user stories is mainly used for the initial development of the application (initial requirements).
